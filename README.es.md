# PASSWORD MASTER — Versión ES

> **Cyber Fortress te necesita.** Defiende la ciudad digital de los hackers mientras aprendes
> ciberseguridad de verdad.

Videojuego *tower defense* educativo. Cada torre es una medida de seguridad real (contraseñas
robustas, MFA, gestores, firewalls, *phishing detection*, SOC). Inspirado en *Kingdom Rush*,
*Brawl Stars* y *Fall Guys*. Stack: React 19 + Node + Express + Prisma + PostgreSQL.

Ver [`README.md`](./README.md) en inglés, [`ROADMAP.md`](./ROADMAP.md) y
[`docs/ARCHITECTURE.md`](./docs/ARCHITECTURE.md).
